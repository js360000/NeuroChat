External marketing kit

NeuroChat standalone HTML asset pack.

These files are independent from the website build and are based on the real NeuroChat frontend language inside neurochat-web: dark-blue theme, Lexend typography, glass surfaces, blue-violet gradients, soft neon glow, and low-stimulation spacing.

Open brand system Open device previews Open campaign assets
Files are self-contained and can be opened directly in a browser or handed to design, growth, or content teams without depending on the app runtime.
Source alignmentTokens, motion feel, and component styling were pulled from neurochat-web/src/index.css and the main feature pages.
Feature focusMessaging, AI explanation, discover, energy tracking, accessibility, and calm/safe interaction states.
Output structureOne file for the brand system, one file for device-led product storytelling, and one file for social and campaign-ready layouts.
